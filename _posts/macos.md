## brew

### 安装

```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

### brew 缓存位置`/Users/${UserName}/Library/Caches/Homebrew`
### brew 安装目录`/usr/local/Cellar/`

### 使用

brew安装jdk

```
brew cask install java
```

brew安装pip

```
brew install pip
```
