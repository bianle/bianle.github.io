发表于 2015-03-26   |   分类于 java   |  

MyEclipse垃圾箱设置

  1. 找到你的MyEclipse的工作空间。路径：<workspace>/.metadata/.plugins/org.eclipse.core.runtime/.settings

  2. 用记事本或写字板打开org.eclipse.ui.prefs添加：SHOW_MEMORY_MONITOR=true

  3. 重启MyEclipse就会看到下面有个小垃圾箱。这个是用来释放MyEclipse内存用的。

linux配置环境变量

发表于 2014-03-25   |   分类于 web前端   |  

在该profile文件中添加：

JAVA_HOME=/usr/java/jdk1.6.0_10 JRE_HOME=/usr/java/jdk1.6.0_10/jre PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin CLASSPATH=.:$JAVA_HOME/lib/jt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib(注意：前面的.: ，linux中的；号为：号) export JAVA_HOME JRE_HOME PATH CLASSPATH
