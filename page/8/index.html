  • vps (阿里云等)
  • php
  • mysql
  • nginx

安装

  • 下载安装包官网

  • 上传到vps(操作系统为Ubuntu的阿里云为例)

  • 配置nginx https://github.com/owncloud/documentation/wiki/ownCloud-9.x-nginx-config

  • 制作自签名SSL证书 给Nginx配置一个自签名的SSL证书
