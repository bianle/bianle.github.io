安装ppp和pptp

配置pptp

wjvpn为自定义连接名
bianle为vpn用户名
192.200.194.142为vpn服务器地址
执行命令系统会提示输入vpn的密码

编辑/etc/ppp/peers/wjvpn

改为:

# written by pptpsetup
pty "pptp 192.200.194.142 --nolaunchpppd"
lock
#noauth
nobsdcomp
nodeflate
name bianle
remotename wjvpn
ipparam wjvpn2
require-mppe-128
refuse-pap
refuse-chap
refuse-eap
refuse-mschap

编辑chap-secrets

改为:

# Secrets for authentication using CHAP
# client server secret IP addresses

# added by pptpsetup for wjvpn bianle wjvpn “xxxxx” *
